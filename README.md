### ♾️ InfinityMath4All (Official Platform)

Welcome to the official repository for **InfinityMath4All**, an open-source, student-led digital platform dedicated to bridging the educational resource gap. We leverage the power of collaborative engineering and advanced mathematics to deliver free, high-quality competitive math (MOEMS, Noetic)  and foundational elementary math enrichment to elementary students.

* * *

### 🚀 The Vision: Our Dual-Impact Model

1.  **The Software & Publishing Engine:** We code our own custom, responsive web application using a modern tech stack and typeset our original competitive math textbooks and problem sets directly in LaTeX.
2.  **The Social Impact Bridge:** We use this unified, digital infrastructure to deliver secure, interactive, and completely free virtual math workshops over Zoom to students in Grades 1–5.

* * *

### 🛠️ Tech Stack & Architecture

This platform is custom-built from scratch to ensure responsiveness, performance, and scalability:

*   **Frontend Framework:** [Vite](https://vitejs.dev/) + [React](https://react.dev/)
*   **Type Safety:** [TypeScript](https://www.typescriptlang.org/) (Strictly typed architecture)
*   **Styling Engine:** [Tailwind CSS](https://tailwindcss.com/)
*   **Curriculum Publishing:** [LaTeX](https://www.latex-project.org/) (Modular mathematical typesetting)

### 🤝 Democratic Contribution Workflow (No Hierarchies)

We operate InfinityMath4All as a completely flat organization. We are all equal Co-Founders with specialized technical and operational focuses. To submit math content or code updates:

1.  **Fork the Repository** to your personal GitHub account.
2.  **Create a Feature Branch** detailing your update (`git checkout -b feature/chapter-1-counting`).
3.  **Commit Your Code Changes** cleanly (`git commit -m "Added MOEMS Venn Diagram problem set"`).
4.  **Open a Pull Request (PR)** targeting the `main` branch. The collective team will review, check for mobile layout bugs/typesetting typos, and merge the code as a unit.

* * *

### 🔒 Safety, Privacy & Compliance (COPPA)

InfinityMath4All treats minor safety as a strict baseline constraint. Our operations enforce:

*   **Absolute Anonymity:** We never collect or store full legal last names of minor students.
*   **Secured Zoom Rooms:** Roster-verified waiting rooms, disabled student-to-student private messaging, and locked screen sharing.
*   **Media Redaction:** All screenshots taken of the digital classroom gallery are subjected to strict face-blurring, cropping, or stylization before public web dissemination.
